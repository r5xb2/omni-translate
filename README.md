# OmniTranslate 網頁即時語音翻譯

## 1. 這個專案是什麼
OmniTranslate 是一個在網頁上運作的即時語音轉文字與翻譯工具。
你只需要打開網頁並對著麥克風說話，系統就會自動將你的語音轉換成文字，並同步翻譯成中文。
完全透過瀏覽器操作，不需要安裝複雜的軟體，是一款適合所有人輕鬆上手的翻譯輔助工具。

## 2. 這個專案可以解決什麼問題
- **原本的痛點：** 當你參與外語線上會議、或是觀看沒有字幕的外國影片時，常常會因為聽不懂或跟不上語速而錯失重要資訊，事後要補做筆記也因為沒有講稿而非常耗時。
- **改善後的體驗：** 透過這個工具，你可以實現「說即翻、聽即翻」的無縫體驗。對話發生的當下，網頁就會自動幫你打出原文的逐字稿並附上流暢的中文翻譯，讓你輕鬆跟上對外溝通的節奏。

## 3. 適合的使用情境
- **跨國線上會議：** 與外國客戶或同事開會時，在螢幕旁同時開啟這個網頁，讓它當你的專屬即時翻譯小秘書。
- **無字幕外語影音學習：** 播放外語 YouTube 影片、Podcast 或是線上課程時，利用電腦收音即時生成翻譯雙語字幕。
- **快速語音筆記：** 當有靈感時，直接用嘴巴說出來，它不僅幫你紀錄，還會順便轉換成中文，這場會議或演講結束後還能一鍵打包匯出對話紀錄。

## 4. 主要功能
- 🎙️ **自動偵測說話**：不需要手動反覆按按鈕，當你開始說話就會自動收音，停頓時便會自動送出並開始翻譯。
- ⚡ **多種轉錄模式**：提供「一般轉錄 (REST)」與「低延遲即時轉錄 (WebRTC)」模式，滿足不同情境的速度與穩定性需求。
- 🌐 **即時中英翻譯**：自動將說出來的外文翻譯成繁體中文（亦可在設定中選擇關閉，僅顯示原文）。
- 📋 **對話紀錄匯出**：網頁上生成的所有對話記錄都可以一鍵複製，或是直接匯出並妥善保存。
- ⚙️ **支援多引擎切換**：可依個人喜好，自由選擇並切換使用 Groq 或是 OpenAI 的人工智慧來進行辨識與翻譯。

## 5. 安裝方式
請先確認你的電腦已經安裝 [Node.js](https://nodejs.org/zh-tw) (建議 20 以上版本)。
打開你的「終端機」(Terminal / 命令提示字元)，依序輸入並執行以下指令：

```bash
# 1. 將此專案下載到你的電腦中
git clone https://github.com/r5xb2/omni-translate.git

# 2. 進入專案資料夾
cd omni-translate

# 3. 安裝必要的套件 (視網路速度約需 1~2 分鐘)
npm install
```

## 6. 設定方式
本專案**不需要**手動修改任何程式碼或環境設定檔 (如 `.env`)。
所有的設定都直接在網頁畫面上進行，簡單且直覺：

1. 啟動網頁後，點擊網頁右下方的「⚙ 齒輪圖示」進入設定畫面。
2. 選擇你要使用的服務商 (推薦初次使用可選免費的 Groq，或者你有綁定信用卡的 OpenAI)。
3. 在欄位中填入你至官網申請好的 **API Key**。
4. 點擊「儲存」即可完成設定。

*(註：你輸入的 API Key 僅會加密並儲存在你的個人瀏覽器中，絕對不會被上傳到其他伺服器，安全有保障。)*

## 7. 操作方式
**啟動專案：**
在終端機中確認你位於專案資料夾內，輸入以下啟動指令：
```bash
npm run dev
```
啟動後，請在瀏覽器 (推薦使用 Chrome 或 Edge) 中開啟網址：`http://localhost:5173`。

**如何操作工具：**
1. 確認已在畫面**下方控制列右側**的「⚙ 設定」中填妥 API Key。
2. 點擊控制列左側的「**▶ 開始**」按鈕。
3. 瀏覽器會彈出「要求使用麥克風」的提示視窗，請點選「**允許**」。
4. 開始說話或將麥克風對準音源，畫面中央會即時顯示原文與中文翻譯。
5. 需要中途停頓時，點擊「**⏸ 暫停**」；繼續時點「▶ 繼續」。
6. 使用完畢後點擊「**■ 停止**」，系統會**自動跳出匯出視窗**，選擇「下載 Markdown (.md)」或「下載純文字 (.txt)」即可儲存紀錄，選「略過」則直接清除。
7. 若有多段對話想即時複製，可在控制列找到「**⎘ 複製全部**」按鈕（有紀錄才會出現）。
8. 結束後，在終端機按 `Ctrl + C` 關閉服務。


## 8. 常見問題
**Q1: 為什麼點擊錄音後，畫面完全沒有反應也沒有出現任何文字？**
A: 最常見的原因有兩個：
1. 忘記在**畫面下方控制列右側**的「⚙ 設定」中填入並儲存 API Key。
2. 瀏覽器的麥克風權限被阻擋了。請檢查網址列最左側的「鎖頭符號」，確認狀態已設為「允許使用麥克風」。

**Q2: 系統會不會偷偷把我的錄音存到雲端？**
A: **不會的。** 系統收到你的語音後，僅會在處理的當下傳送給 OpenAI 或 Groq (依據您在第一步的選擇) 將其轉換為文字，隨即銷毀。本專案不具備且伺服器端也沒有任何會存檔或側錄語音的設計。

**Q3: 原文翻譯出來的中文看起來怪怪的，有時候還會出現簡體字？**
A: 這是因為某些國外的 AI 引擎預設學習資料包含較多簡體中文。系統內已經內建自動轉換繁體中文的機制來修正，但若涉及專門技術名詞偶爾會有落差。

**Q4: 我應該選擇 Groq 還是 OpenAI？**
A: 如果你是因為好用想先試試看，強烈建議免費註冊取得 **Groq** 的金鑰 (因為完全免費，且翻譯給字極快)。如果這對你的工作非常重要，需要最頂尖精準的語音辨識度，推薦使用 **OpenAI**。
